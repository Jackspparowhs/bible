FAQ

Do I need an API key?

Not for the demo map. If you add a paid aircraft-data API, you will need the key ‚Äî this UI is ready to accept that client-side or via a small proxy.

Is the map live?

The map page loads immediately and can poll an API for live positions; demo markers are static so it works without any backend.

Can I search by registration?

Yes ‚Äî the search box in the map UI accepts registration or callsign and will zoom to results once hooked to data.
